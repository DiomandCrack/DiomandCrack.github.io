全局对象

每个JavaScript环境都有一个全局对象，每当浏览器加载一个新页面的时候，它都会创建一个新的全局对象，并给它附上一些属性和方法。

在浏览器窗口中,这个全局对象通常用window来表示

数组指的是一组带有有序编号的值的集合

3.数组里面可储存任意类型的数据

包装对象

1.字符串和数字不是对象，但是有属性和方法

2.ECMAScript 5.1规范中 明确指出 toString 方法 null和undefined都能转换成字符串 但在浏览器实现中却报错了

JavaScript 将数字和字符串临时转换成了对象，这个对象有附带一些的属性和方法，一旦属性和方法结束调用，立即被销毁，这一类对象被称为包装对象。由于null和undefined不能转换成对象所以报错。

在读取字符串、数字和布尔值的属性值(或方法)的时候,表现的像对象一样。但如果视图给其属性赋值，则会忽略这个操作，修改只是发生在临时对象上，而这个临时对象并未继续保留下来
